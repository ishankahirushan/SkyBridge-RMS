## 📋 About

This is a university level software engineering project designed to simulate the workflow of a real airline ticket agency. The system showcases modern operational processes including flight searching, passenger verification, booking management, payment processing, ticket generation, refunds, and analytics reporting while maintaining a clean enterprise style architecture. Built entirely with core web technologies, the project focuses on practical learning, realistic business logic, and full stack development fundamentals.

## Testing

Run Phase 16 booking registry E2E tests with Playwright:

```bash
npm install
npx playwright install
npm run test:e2e
```

Note: start the PHP app server first at `http://localhost:8000`.
